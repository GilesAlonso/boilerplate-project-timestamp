Timestamp Microservice

Convert a date into a Unix timestamp and UTC format.

Example Usage:

  • [project url]/api/2015-12-25
  • [project url]/api/1451001600000

Example Output:

{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}

Whoami Microservice

Get details about your IP address, language, and software.

Example Usage:

Access the API by visiting: [project url]/api/whoami

Example Output:

{
  "ipaddress": "192.168.0.1",
  "language": "en-US",
  "software": "Mozilla/5.0"
}
